## Dagster docs audiences

- **Data pipeline builders:** People responsible for creating, configuring, and maintaining data pipelines for their organization. This audience uses the Dagster framework and features to build, automate, operate, log and debug, observe, and test data pipelines. Assume basic familiarity with Python and SQL.
- **Data platform owners:** People responsible for setting up, configuring, and troubleshooting the data platform in their organization that runs the pipelines created by data pipeline builders. Assume basic familiarity with DevOps and cloud deployment concepts.
- **Economic buyers:** People with purchasing power who are evaluating the Dagster platform and framework for integration into their organization.
- **Open source contributors:** People contributing code and docs to the [Dagster OSS project](https://github.com/dagster-io/dagster/). Some OSS contributors use open source Dagster and some use Dagster+. Assume moderate to advanced knowledge of Python.
