Dagster+ AI helps you detect, understand, and resolve pipeline failures. It combines conversational debugging, automatic failure detection, and AI-powered code fixes into a single workflow.

## Capabilities

- **[Chat with Dagster+ AI](/guides/labs/dagster-ai/chat)** — Summarize run failures, identify root causes, and ask follow-up questions in a conversational interface.
- **[Proactive monitoring](/guides/labs/dagster-ai/proactive-monitoring)** — Dagster+ AI periodically scans your deployments and creates [Issues](/guides/labs/issues) for detected failure patterns.
- **[Auto-implement with AI](/guides/labs/dagster-ai/auto-implement)** — Dispatch an AI coding agent from any Issue to create a fix as a GitHub pull request.
